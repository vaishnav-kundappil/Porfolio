Electric Motor Prediction

This machine learning project focuses on predicting the speed of an electric motor based on various input parameters. Built with Python and Streamlit, it provides an interactive interface where users can enter motor characteristics and obtain real-time speed predictions. A neural network-based model powers the predictions, leveraging deep learning techniques to analyze motor behavior and estimate speed with high accuracy.
